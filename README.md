- Node.js (v18+)

## 🔧 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/doomscroll.git
   ```

2. Install dependencies
   ```bash
   cd doomscroll
   npm install
   ```

3. Start the development server
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📝 Usage
